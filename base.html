BASE OBJECT

MASUKAME

MASUKAMEのBASE OBJECTは、
日本産のヒノキを使用し、完全受注生産で制作される工芸オブジェクトです。

素材には、日本で育ったヒノキ材を用いています。
枡部分は、日本の熟練した職人の手によって、
一点ずつ手作業で仕上げられます。

枡以外のパーツは、
日本の木材を用いた表現を専門とする
木材アートデザイナーによって制作されています。

それぞれの工程は分業ではありますが、
最終的には一体の作品として完成することを前提に設計・制作されています。

MASUKAMEは工業製品ではありません。
すべてが手作業で行われるため、
木目や仕上がりには一体ごとの個体差があります。

各作品には、
シリアルナンバー入りの証明書が付属します。

仕様概要

素材 日本産ヒノキ
制作 日本の熟練職人 ／ 木材アートデザイナー
制法 一点ずつ手作業
生産方式 完全受注生産
付属 シリアルナンバー入り証明書

名入れ・金箔仕上げなどのカスタマイズも承ります。
